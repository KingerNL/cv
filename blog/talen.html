    De programmeer talen die ik heb geleerd

    Hoewel het meevalt hoeveel programmeer talen er zijn die ik ken /heb geleerd. Dacht ik toch dat het een goed idee was om te praten over de talen die ik wel ken, hoe goed ik ze ken, mijn ervaring ermee en eventuele projectjes.

    - Arduino

    Hoewel Arduino uit zichzelf geen taal is maar een variant van C*. Vind ik het makkelijker om Arduino te zeggen aangezien de meeste mensen het merk wel kennen maar niet de taal. De Arduino omgeving van mijn eerste stap in het coderen. Ik heb hier de basis bezet voor bijna alles wat volgde. Was er een school projectje wat ik snel in elkaar moest zetten; Arduino. Als je een beginner bent en je eerste stappen wil zetten in Hardware-programmeren is Arduino een aanrader.

    - CSS / HTML

    Ik stop deze 2 bij elkaar aangezien ze voor mij gepaard gaan. Ik begon met echt CSS/HTML toen ik erachter kwam hoe belangrijk een CV is om in het echte leven aan werk te komen, ik zie een CV als een eerste ontmoetingen. Ik ben na een hoop zoekwerk over hoe je het best een CV kan schrijven gaan kijken naar alternatieve oplossingen, en kwam tot de conclusie dat een site mijn creativiteit combineert met het codeer vermogen wat ik toen nog niet had. Na de basics te leren kwam ik steeds verder, had ik een idee? Zoek het op, zoek naar het antwoord, en leer. Als je een site wil maken kan je er bijna niet omheen om dat zonder CSS/HTML te doen. Ik vind mezelf redelijk ervaren en kan een gemiddelde site in elkaar zetten. Het beste voorbeeld van mijn werk is de site waar je dit nu in leest.

    - Linux

    Na het maken van mijn eigen site”, ben ik wat dieper in het daadwerkelijke computer besturing ’s systeem gaan duiken (OS in het kort). Ik heb geleerd hoe je in een kernel mappen kan maken, documenten kan aanmaken/aanpassen, navigeren en tot slot heb ik geleerd over het mappenstructuur van je OS systeem. Na enig tijd heb ik geleerd over Git & Github en hoe je daar mee kan werken in een kernel. Voor mensen die wat meer willen weten over Git & Github is het een goede instap om eerst te kijken wat een kernel is en hoe het werkt.

    - Python

    Een van de belangrijkste programmeer talen die ik kan: Python, als laatst. Ik ben begonnen met Python te leren omdat ik tijdens mijn Minor AI (Artifficial Intelligents) moest werken in Python, zonder enige voorkennis van Python ben ik hevig wat voorwerk gaan doen in de vakantie voor mijn minor. En ik durf nu te zeggen dat Python mij redelijk licht. Hoewel ik nog niet een Python “guru” ben durf ik met mijn voorkennis mijn minor te doen. Ik ben nog niet begonnen met deze minor dus kan er helaas nog niet veel over zeggen, je kan mij (in dien geïnteresseerd) altijd een bericht te sturen als je vragen hierover hebt.

    - Honorable mentions:

    Siemens : Hoewel Siemens niet zo zeer programmeren is dacht ik het toch te vermelden als een omgeving waar ik in heb gewerkt.

    Kali / Ubuntu : Hoewel ik minder hiermee werk, weet ik wel hoe deze besturingssystemen werken.
